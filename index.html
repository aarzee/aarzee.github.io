Projects

Lithium

A tweak for jailbroken iOS, Lithium allows users to customize the battery indicator on their device. It uses JavaScript and HTML5 Canvas to dynamically generate images for any battery state and device resolution. Lithium organically achieved 200,000 downloads through Cydia.

Communique

Communique is a full-stack web app using Ruby on Rails on the backend and React/Redux on the frontend. A clone of Medium, it offers rich text editing with image upload, comments, follows, a feed, and likes.

Natrix

Natrix is a barebones JavaScript DOM manipulation library. It provides useful functionality for traversal and editing.

Flexis

Flexis is a clone of Tetris that can be played in the browser, written in vanilla JavaScript.

6kk

6kk is a Python script allowing users to play through the menu minigames of SUPERHOT through the Windows Command Prompt. It was featured in PC Gamer and Kotaku Australia.
